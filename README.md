# embedded-systems-projects

A collection of embedded systems projects built while relearning core engineering best practices. 
Wokwi is used as the primary simulator for its broad microcontroller support and seamless VS Code integration for local compilation. (wokwi.com). 
Claude and other AI is used as a learning assistant throughout these projects — because using powerful tools effectively is part of good engineering.
